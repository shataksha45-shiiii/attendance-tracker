# attendance-tracker

A minimalist attendance management Progressive Web App designed for academic discipline tracking.

## Technical Stack

* Frontend: React.js
* Backend: Firebase
* Concepts: Data Visualization, Analytics, PWA Architecture

## Features

* Attendance Tracking: Log daily class participation with local-first persistence.
* Dynamic Analytics: Real-time calculation of attendance percentages and safe-to-bunk limits.
* Synchronization: Secure cloud sync for historical data logging and cross-device access.
* Offline Support: Service worker implementation for reliable performance in low-connectivity environments.

## Installation

1. Clone the repository.
2. Execute `npm install` to load dependencies.
3. Execute `npm run dev` to initialize the local development server.
